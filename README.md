# FreshMart POS 🛒

A single-file grocery store management system built with HTML, CSS, and JavaScript.

## Features
- 🔐 Login Authentication (Admin/Cashier)
- 🛒 Real-Time Point of Sale
- 📦 Product & Inventory Management
- 📊 Sales Analytics & Reports
- 🌙 Dark Mode
- 🔍 Search & CSV Export
